# Analysis of the Gateway Hypothesis Using NSDUH Data

This repository contains applied data analysis code exploring the gateway hypothesis using publicly available NSDUH (National Survey on Drug Use and Health) data for the United States.

The project focuses on data preprocessing, exploratory analysis, and statistical modeling implemented in R.  
All code in this repository was authored by me, and all data sources used are public.

---

## Overview

The project includes:
- data cleaning and transformation pipelines
- preparation of analysis-ready datasets
- statistical modeling and exploratory analysis related to substance use patterns

The work is derived from academic research interests but is presented here as an **independent applied analysis**.

---

## Technical Information

- **Programming language:** R
- **Raw dataset:** `data/NSDUH_2022.Rdata` (publicly available at https://www.samhsa.gov/data/data-we-collect/nsduh-national-survey-drug-use-and-health/datafiles/2022)
- **Data transformation scripts:** `code_r/kod_dane_wip.R`
- **Processed dataset:** `data/NSDUH_2022_dataset_arch.csv`
- **Bibliography:** `paper/refs.bib`
